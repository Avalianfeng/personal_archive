# 03B-核心档案目录.md



> 产物 B · 文件一:核心档案(稳定层)

>

> 更新频率:数月到数年一次,仅在长期特征有实质变化时修改。

> 目的:「这个人是谁」的骨架 — **事实(经历)为主体,人格(特征)在其上**。

>

> 娱乐偏好、审美、消费观等长期稳定倾向,记入 §3「长期稳定的价值倾向」条目。

> 技能、健康、生活习惯等补充信息,留给日后插件档案,主档案不重复。



------



# 0. 档案头



- 建档时间

- 当前版本

- 建档动机(一句话)

- 完整度自评:高 / 中 / 低



------



# 1. 基础信息



- 年龄阶段、教育、职业、居住地

- 重要身份标签

- 成长背景(一段,50 字以内)

- 一句话画像(观察者视角)



------



# 2. 人生经历



> 事实主体章。每段:事实直述 + 时间/阶段;不写推断。



## 成长背景



地域、家庭氛围、童年至青年关键环境(已知范围内)



## 教育经历



学历路径、专业、关键学业节点



## 职业与工作



行业、岗位、重要职业节点与转折



## 重要转折与迁移



搬家、换城、换轨、重大人生节点(与时间线呼应)



------



# 3. 核心特征



## 长期稳定的思维方式



## 长期稳定的价值倾向



(含娱乐偏好、审美、消费观等长期条目)



## 长期稳定的关系模式



## 已确认的边界与禁区



(含深度内容,以「自述:」标注来源)



------



# 4. 重要人物



姓名 / 关系 / 一句备注 / 与本人的关系状态



------



# 5. 原始语料索引



5–10 条最有档案价值的原话 + 所在章节。不全文粘贴 intake。



------



# 6. 档案关键词



5–8 个,用于快速定位这个人。



------



# 注意事项



1. 档案优先保存事实;分析性长段压缩为条目。§2 人生经历是事实主体,§3 核心特征是推断性归纳。

2. 每条信息须区分来源:事实直述 / 「自述:」/ 「观察:」。

3. 缺料写「尚未记录」,不阻断。

4. 不用 MBTI 标签;类型学仅附录可选且标注「参考」。
