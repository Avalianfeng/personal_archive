```text
        ↓ build_questions.py

```

---

## uid 与 id

| 字段 | 角色 |
| --- | --- |
| `uid` | **主键**（8 位 hex）；`related` / 未来映射引用 uid |
| `id` | 人类可读编号 `Q-SELF-001`；**删题不重排**，允许空档 |
| `order` | 可选展示顺序，与 id 解耦 |

Builder 缺失 `uid` 时自动生成并写回 md（`--no-write-uid` 可关闭）。

---

## 系统不可提问（system_unaskable）

题干预设**在场对话者**（采访者↔被采访者），如「你最喜欢**我的**哪段回忆？」。整理阶段**不入 categories**，写入 [`rejected/system_unaskable.md`](../rejected/system_unaskable.md)。

## raw 队列

```text
question_sources/   来源档案馆（卡片，不移动）
raw/pending/        待整理
raw/processed/      已整理完成
```

## 两层生命周期

| 维度 | 载体 |
| --- | --- |
| **文件** | raw/pending · raw/processed · rejected/ · duplicates/ |
| **题目** | frontmatter `status`: active · candidate · deprecated |

---

## frontmatter 示例

```yaml
---
uid: 9d7e2c5a
id: Q-SELF-001
category: self
subcategory: 他人评价
type: open
interaction: reflection
source: StoryCorps
tags:
  - self_image
depth:
validation: false
related: []
status: active
---

别人最容易误解你的地方是什么？
```

**勿用**单独一行 `---` 作 Markdown 分隔线。

---

## 字段（v0.3）

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| `uid` | Builder 补 | 8 位 hex 主键 |
| `id` | 是 | 人类编号；删题不重排 |
| `category` | 是 | 与文件名 slug 一致 |
| `subcategory` | 是 | |
| `type` | 是 | open · single · multi · scale · sort · fill · agreement |
| `status` | 是 | active · candidate · deprecated |
| `interaction` | 建议 | story · self_report · scenario · comparison · rating · reflection · future |
| `source` | 建议 | |
| `tags` | 否 | 检索用，非 mapsTo |
| `depth` | 否 | 可空 |
| `related` | 否 | 题号或 uid |
| `order` | 否 | 展示顺序 |

**禁止**：difficulty · mapsTo · dimension · trait
