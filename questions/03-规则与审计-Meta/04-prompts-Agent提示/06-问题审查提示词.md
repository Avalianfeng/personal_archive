# 问题审查 Agent · 提示词

> **阶段**：**清理期启用** — 收集期不跑。启动见 [04-清理期Agent启动.md](./04-清理期Agent启动.md)。  
> **用法**：读查重建议 + `qcli`（默认 JSON 输出）；**禁止**改 categories md。

## 输入

- 查重 Agent 输出的建议命令块
- `03-generated-审计产物/duplicate_report.md`（若已跑 `--with-duplicate-scan`）
- `python questions/qcli.py show --ids Q-XXX,Q-YYY --fields text,subcategory,type,source`

## 输出（qcli 命令）

```bash
python questions/qcli.py relate Q-SELF-012 --duplicate-of Q-SELF-001
python questions/qcli.py edit Q-SELF-012 --set status=deprecated
```

（qcli 默认 JSON；人类终端加 `--human`。）

## 禁止

- 直接编辑 `02-问题地图-Views/*.md`
- bulk 导入 `01-文献与来源-Library/`
