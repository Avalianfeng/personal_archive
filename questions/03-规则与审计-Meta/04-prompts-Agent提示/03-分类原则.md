# 分类原则

> **功能**：人类速查 — 一级分类的定义、子主题、例题与邻类边界。
> **关联**：[00-整理规范-v1.0.md](../00-整理规范-v1.0.md) · [02-问题整理提示词.md](./02-问题整理提示词.md) · [02-问题地图-Views/](../../02-问题地图-Views/README.md)

---

## 总原则

1. **按「这道题主要问什么」归类**，不是按来源（量表、访谈、intake 批注等来源不作目录分类）。
2. **诚实度不是主题分类**。校验题设 `validation: true`，归入其内容所属类。
3. 一道题只能有一个一级类；子分类（`##`）可按需新增。
4. 本阶段只整理题干，**不分析人格、不推断、不 mapsTo**。
5. **整理 Agent 禁止读取** [02-维度地图-dimensions.md](../../02-维度地图-dimensions.md) 与档案层文件。

---

## 一级分类冻结

至少在题库达到 **500–1000 题**前，**不要新增一级分类文件**。

| 允许 | 不允许 |
| --- | --- |
| 在现有文件下新增 `## 子分类` | 新增 `依恋问题.md`、`创伤问题.md`、`人格问题.md` 等心理学理论类文件 |

当前分类是**问题功能分类**，不是心理学理论分类。

---

## 来源优先级（人类参考，非 Agent 自动跳过）

整理来源时，人类可按以下优先级安排批次（详见 [00-整理规范-v1.0.md §来源优先级](../00-整理规范-v1.0.md#来源优先级人类排期参考)）：

| 优先级 | 来源类型 | 归类提示 |
| --- | --- | --- |
| **P1** | 人生故事访谈、StoryCorps、口述史、AAI | 多为 `story` 回答形式；现实 / 情感 / 自我认知 |
| **P2** | PVQ、RSES、WHOQOL 等 | 价值 → `价值问题`；自尊 / 生活质量 → `状态问题` 或 `自我认知` |
| **P3** | Big Five、MBTI | 多为 `rating` / `认同度`；分散入各类，信息增量有限 |

**量表 / 筛查**（PHQ-9、GAD-7 等）：可收录，优先入 **状态问题**，非核心来源。

**检索维度**（非分类）：`type` · `interaction` · `tags` · `subcategory` · `status` — 见 [02-schema-格式契约/01-格式规范.md](../02-schema-格式契约/01-格式规范.md)。分类文件是人类浏览轴；tags 是跨类搜索轴。

## 现实问题

**回答什么**:发生过什么、是什么、客观经历与事实。

| 典型子主题 | 示例 |
| --- | --- |
| 家庭与成长 | 小时候最常相处的人是谁？ |
| 教育 | 大学专业是如何选择的？ |
| 职业与经历 | 有没有长期坚持超过三年的事情？ |
| 时间线与事件 | 过去五年里，对你影响最大的一件事是什么？ |

**与邻类边界**:

- vs **状态**:现实问「发生过」，状态问「现在怎样」。
- vs **自我认知**:现实问客观事实，自我认知问「你怎么看自己」。

---

## 情感问题

**回答什么**:感受什么、情绪体验、关系中的主观感受。

| 典型子主题 | 示例 |
| --- | --- |
| 亲密关系 | 你更害怕被拒绝还是被忽视？ |
| 友谊 | 朋友爽约时你通常什么感受？ |
| 孤独与依恋 | 独处时你更常感到平静还是不安？ |
| 情绪表达 | 谈到自己的感受时，你通常很难开口还是很容易表达？ |

**与邻类边界**:

- vs **关系行为题**:若主要问「你会怎么做」→ 决策；主要问「你感受如何」→ 情感。
- vs **状态**:情感可问一贯倾向；状态强调近期、当下。

---

## 决策问题

**回答什么**:怎么选择、怎么判断、面临取舍时的倾向。

| 典型子主题 | 示例 |
| --- | --- |
| 风险偏好 | 稳定但无聊的工作 vs 有风险但有趣的项目，你更倾向哪个？ |
| 信息依赖 | 做重要决定前，你更常独自分析还是咨询他人？ |
| 速度与风格 | 你通常快速决断还是反复权衡？ |
| 情境取舍 | 收入翻倍但不喜欢的工作，和喜欢但收入一般的工作，你会选哪个？ |

**与邻类边界**:

- vs **价值**:决策问「你会怎么选」；价值问「什么对你重要、什么不可接受」。
- vs **现实**:决策是假设或倾向题；现实是已发生的事实。

---

## 状态问题

**回答什么**:现在怎么样 — 近期压力、睡眠、动力、困扰、身心状态。

| 典型子主题 | 示例 |
| --- | --- |
| 压力与焦虑 | 最近最困扰你的事情是什么？ |
| 睡眠与精力 | 过去一个月，你的睡眠质量如何？ |
| 动力与方向 | 你现在对日常事务的动力感如何？ |
| 近期情绪基调 | 最近两周，你的整体情绪基调更接近哪一端？ |

**与邻类边界**:

- vs **情感**:状态强调**当下/近期**；情感可含一贯的情绪模式。
- vs **现实**:状态是当前主观状态；现实是过去客观事实。

---

## 自我认知

**回答什么**:怎么看自己 — 自我描述、自我评价、他人眼中的自己。

| 典型子主题 | 示例 |
| --- | --- |
| 自我描述 | 用三个词形容你自己。 |
| 他人误解 | 你觉得别人最容易误解你的地方是什么？ |
| 自我欣赏与不足 | 你最欣赏自己的什么？你最想改进的是什么？ |
| 角色认同 | 在朋友、家人、同事面前，你觉得自己分别是怎样的人？ |

**与邻类边界**:

- vs **价值**:自我认知问「我是谁/我怎么看自己」；价值问「什么重要/什么不能接受」。
- vs **现实**:自我认知是主观看法；现实是客观经历。

---

## 价值问题

**回答什么**:认为什么重要、底线是什么、优先级与原则。

| 典型子主题 | 示例 |
| --- | --- |
| 优先级 | 如果必须在事业、关系、自由中选一个优先，你会选哪个？ |
| 底线与原则 | 你最不能接受别人什么行为？ |
| 道德取舍 | 什么情况下撒谎是可以接受的？（若无接受项则标注开放） |
| 人生目标 | 对你来说，「成功」意味着什么？ |
| 个人价值观 | PVQ 式 portrait 陈述 +「像我」量表（如：想出新颖点子对你有多重要？） |

**与邻类边界**:

- vs **决策**:价值问「什么重要」；决策问「具体情境下怎么选」。
- vs **自我认知**:价值是原则与优先级；自我认知是自我形象。

---

## 其他

**何时使用**:暂无法归入以上六类，或跨类过强、需人工再判。

处理建议:放入 `其他.md`，在题目标题旁简短注明「待定类」；积累到一定数量后由人类决定是否新增一级类。
