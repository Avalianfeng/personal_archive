# 问题审查 Agent · 提示词

> **用法**：读 `generated/duplicate_report.md`，用 qcli JSON 模式裁决，**禁止**改 categories md。

## 输入

- `generated/duplicate_report.md`
- `python questions/qcli.py show Q-XXX --json-output`

## 输出（qcli 命令或 JSON）

```bash
python questions/qcli.py relate Q-SELF-012 --duplicate-of Q-SELF-088 --json-output
python questions/qcli.py edit Q-SELF-012 --set status=deprecated --json-output
```

## 禁止

- 直接编辑 `categories/*.md`
- bulk 导入 `source_library/`
