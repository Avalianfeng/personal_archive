# pending · 待整理 raw

新采集的原始材料放这里。整理完成后**移动**到 [../processed/](../processed/)（不删除）。

## 命名

文件名以 `[Px]` / `[批次]` / `[元]` 开头，与 [question_sources/](../../question_sources/README.md) 卡片优先级一致。资源管理器按名称排序时 P1 自然靠前。

| 前缀 | 何时用 |
| --- | --- |
| `[P1]` `[P2]` `[P3]` | 已有来源卡片，优先级已定 |
| `[批次]` | `问题库-NNN.md` 等手工 dump，待确认来源 |
| `[元]` | 链接索引，不送整理 Agent |

登记来源：在 `question_sources/` 新建或更新 `[待整理][Px]` 卡片，raw 路径字段指向本目录文件。
