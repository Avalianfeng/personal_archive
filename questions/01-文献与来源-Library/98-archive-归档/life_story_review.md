| 那么，你的故事是什么？                                       |      |
| ------------------------------------------------------------ | ---- |
| 丹·麦克亚当斯在研究过程中发现，每个人都有自己的故事——但要概括一个人的一生并非易事。为了简化这个过程，他设计了一种访谈形式，鼓励受访者分享他们生命中最重要的一些时刻。想要构建你自己的人生故事，不妨参考麦克亚当斯的这份精简版访谈脚本（他建议你和朋友或亲戚一起尝试，轮流讲述和倾听）。最终，你或许会发现，你的人生的确是一个精彩的故事。第一步：人生篇章把你的人生想象成一本书。把故事分成几个章节（至少两三个，最多七个）。给每个章节起个名字，并简要概括其内容。把这看作是你故事的整体大纲。第二步：关键事件在接下来的部分，请思考你人生中的关键事件——具体的时刻，而不是持续数天或数月的经历。对于以下每个事件，请描述你当时的感受或想法，涉及哪些人，以及该事件体现了你怎样的性格：**巅峰体验：**你感受到极度积极情绪的时刻。它应该是你记忆中最美好的场景之一。**人生低谷：**人生故事中的低谷，感到绝望、内疚或痛苦的时刻。**转折点：**你经历了一次重大改变的事件，尤其是一个让你对自己的认识发生转折的时刻。**最早的记忆：**你能辨认出发生了什么事、涉及哪些人以及你当时的想法和感受的最早记忆。**童年重要场景：**一段对你而言意义非凡的记忆。它反映了你当时的哪些特质？**青少年时期的重要事件：**你青少年时期发生的一件特别重要或意义非凡的事情。**重要成人场景：**成年时期发生的某个令人印象深刻的特定事件。**另一个重要场景**第三步：人生挑战回顾过往经历，请描述你人生中遇到的最大挑战。你是如何应对的？它对你的人生轨迹产生了怎样的影响？第四步：角色塑造描述一下你故事中影响最大的正面人物和影响最大的负面人物。（把他们想象成你书中的英雄和反派。）第五步：未来首先，描述一个积极的未来：你希望余生如何发展（尽可能现实一些）。然后，描述一个消极的未来：一些*可能*发生但你希望不会发生的事情。第六步：人生主题回顾你的一生，你能否从中发现一个中心主题、信息或理念？你人生故事的主要主题是什么？ |      |
