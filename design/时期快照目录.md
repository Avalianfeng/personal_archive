# 03B-时期快照目录.md

> 产物 B · 文件二:时期快照(动态层)
>
> 更新频率:每次建档 / 每次追问后更新。
> 目的:「这个人现在在哪里」。
>
> 当前兴趣、当前消费重点等,记入 §1「当前处境」。

------

# 快照信息

- 快照日期
- 距上次更新:X 天(首次建档写「首次」)
- 本次更新触发原因

------

# 1. 当前处境

- 生活状态(一句)
- 当前关注重点
- 当前关键词(3–5 个)
- 当前主要压力源
- 当前兴趣、当前消费重点(如有)

------

# 2. 当前主要张力

## 内在张力

自述 + 观察

## 外在张力

自述 + 观察

------

# 3. 近期重要事件

事件 + 日期 + 一句影响(同时应新增到核心档案 §2「重要转折与迁移」)

------

# 4. 正在探索的问题

列条目,标注「本人知晓」/「本人尚未命名」

------

# 5. 本次建档新增 / 修订

对核心档案的修订建议(供人工确认后写入)

------

# 注意事项

1. 每条信息须区分来源:事实直述 / 「自述:」/ 「观察:」。
2. 禁止把填写者的自我解读直接写成客观陈述。
3. 缺料写「尚未记录」,不阻断。
